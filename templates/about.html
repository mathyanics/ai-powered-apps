About AI Learning Hub

AI Learning Hub is a comprehensive AI-powered platform that enables intelligent interaction with multiple data sources and provides interactive coding practice with personalized feedback.

Core Capabilities:

  • Dataset Q&A: Upload CSV, Excel, or JSON files and query them using natural language. The system automatically converts your questions to SQL and provides formatted results with data visualizations.
  • Document Q&A: Extract insights from PDF and PowerPoint documents using advanced RAG (Retrieval-Augmented Generation) with FAISS vector store. Receive answers with source citations and relevance scores.
  • YouTube Q&A: Analyze YouTube video transcripts and ask questions about the content. View embedded videos alongside AI-generated answers based on caption analysis.
  • AI Coding Exercise: Generate custom programming challenges tailored to your needs, execute code in 13 programming languages (Python, JavaScript, TypeScript, Java, C++, C#, Go, Rust, PHP, Ruby, Kotlin, Swift, R) using cloud-based Piston API, validate your solutions with instant AI feedback, request hints when stuck, and view detailed solutions with explanations.
  • Video Interview: Practice for real job interviews with AI-powered mock sessions. Record video responses with webcam, use real-time speech-to-text in 15 languages (English, Spanish, French, German, Italian, Portuguese, Japanese, Korean, Chinese, Hindi, Arabic), answer role-specific questions (AI Engineer, Product Manager, Software Engineer, Data Scientist, etc.), and receive comprehensive performance analysis with overall/technical/communication scores, strengths, improvement areas, question-by-question feedback with video playback, and hiring recommendations.

Technologies Used:

  • Flask 3.0.0 - Modern Python web framework with session management
  • LangChain & HuggingFace - LLM orchestration and embeddings for semantic search
  • FAISS - Vector similarity search for document retrieval
  • Pandas & SQLite - Data manipulation and in-memory database queries
  • Piston API - Cloud-based sandboxed code execution for 13 programming languages
  • Web Speech API - Browser-native real-time speech-to-text with 15 language support
  • MediaRecorder API - Browser-based video/audio recording for interview sessions
  • KaTeX - Mathematical expression rendering
  • Bootstrap 5 - Responsive UI framework with modern design
  • YouTube Transcript API - Automatic caption extraction
  • PyPDF2 & python-pptx - Document processing libraries

Why AI Learning Hub?

This platform combines the power of large language models with practical data analysis, multi-language coding education, and AI-powered interview preparation. Whether you're exploring datasets, extracting knowledge from documents, learning from videos, practicing programming in 13 languages with real-time execution, or preparing for job interviews with video recording and comprehensive feedback, AI Learning Hub provides an intuitive interface with intelligent assistance every step of the way.

Key Features:

  • 5 Interactive Modules: Dataset QnA, Document QnA, YouTube QnA, Coding Exercise, Video Interview
  • 13 Programming Languages: Run code in Python, JavaScript, TypeScript, Java, C++, C#, Go, Rust, PHP, Ruby, Kotlin, Swift, R
  • 15 Speech Languages: Interview transcription in English (US/UK), Spanish (Spain/Mexico), French, German, Italian, Portuguese (Brazil/Portugal), Japanese, Korean, Chinese (Simplified/Traditional), Hindi, Arabic
  • AI-Powered Analysis: Comprehensive feedback for code submissions and interview performance with detailed scoring
  • Real-Time Execution: Cloud-based code execution via Piston API with instant results
  • Video Recording: Browser-based webcam recording with playback for interview review
